## Per-market contracts

Each market created via the factory deploys three additional contracts:
- AirToken (airToken)
- AirToken (airUsd)
- EXNIHILOPool

These addresses are emitted in the `MarketCreated` event and can be queried via `factory.allPools(index)`.
